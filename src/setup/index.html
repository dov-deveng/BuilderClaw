Connect to Claude

BuilderClaw is powered by Claude AI. You'll need an API key from Anthropic. Your key stays on your computer and is never shared.

How to get your API key:
1. Go to console.anthropic.com
2. Sign up or log in
3. Click "API Keys" in the sidebar
4. Click "Create Key" and copy it
